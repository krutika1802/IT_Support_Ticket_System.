# IT Support Ticket Management System

This project is a simple IT support ticketing system built using Excel to simulate real-world incident management processes.

The system allows tracking of user-reported issues, prioritizing tickets based on severity, and monitoring their status from creation to resolution.

## Key Features
- Ticket logging with unique Ticket IDs
- Priority classification (High, Medium, Low)
- Status tracking (Open, In Progress, Resolved)
- Issue categorization and resolution tracking
- Basic dashboard for ticket overview

## Purpose
The goal of this project is to demonstrate foundational skills required for a Technology Support Engineer role, including issue tracking, prioritization, and structured problem management.

## Skills Demonstrated
- Incident management
- Problem tracking and organization
- Basic data handling using Excel
- Understanding of IT support workflows

This project reflects practical learning and hands-on experience in managing support tickets in a structured environment.
